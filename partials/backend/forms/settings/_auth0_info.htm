How to retrieve your credentials

This is an Auth0 specific handler using the OIDC protocol. You will need to setup an application with Auth0.

  1. Go to https://manage.auth0.com/dashboard and create a new application. Name it something like my-awesome-website
  2. Get the application's Auth0 Domain
  3. Get the application's Client ID
  4. Get the application's Client Secret
  5. Set the application's Valid OAuth redirect URIs to
